## 🎵 About The Project

**Wavify** is a modern music streaming web application that delivers a smooth, engaging, and visually appealing music experience.

Inspired by popular streaming platforms, Wavify allows users to browse songs, discover artists, create playlists, save favorite tracks, and enjoy a beautiful responsive interface.

### ✨ Highlights

- 🎧 Interactive Music Player
- 🔍 Real-Time Search
- ❤️ Liked Songs Collection
- 📂 Playlist Management
- 📈 Recently Played Songs
- 🎤 Popular Artists
- 🔥 Trending Music
- 🎼 Featured Playlists
- 🎵 New Releases
- 💾 Local Storage Support
- 📱 Fully Responsive Design
- ⚡ Smooth Animations
